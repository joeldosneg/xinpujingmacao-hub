# xinpujingmacao-hub

本仓库用于归档与发布多个独立 HTML 页面。这些页面可视为静态资源集合，不绑定任何特定域名或网站。

## 项目简介

这是一个通用型 HTML 页面归档仓库，主要用于存放一系列自包含的网页文件。每个页面均为独立单元，不依赖于外部数据库或后端服务。适合用作页面展示、原型保存或轻量级内容分发。

## 目录结构

```
├── pages/        # 存放所有 HTML 页面文件
├── assets/       # 页面引用的静态资源（CSS、JS、图片等）
├── index.html    # 仓库首页入口（可选）
└── README.md     # 本文件
```

- `pages/`：每个 HTML 文件对应一个独立页面，文件名即页面标识。
- `assets/`：按需存放页面共用的样式、脚本或媒体文件。

## 页面归档说明

- 所有页面均为静态 HTML，兼容主流浏览器。
- 页面间无相互依赖，可单独打开或部署。
- 文件命名建议使用英文小写字母与连字符，便于管理。
- 若页面需要外部资源（如字体、图标），请一并放入 `assets/` 目录，避免引用外部链接。
